# QueueCure

A real-time clinic queue management platform built using React, Express.js, MongoDB, Socket.io and AI-powered wait time prediction.

## Features

- Real-time queue updates
- Reception dashboard
- Patient waiting screen
- Doctor dashboard
- JWT Authentication
- Analytics
- AI Wait Time Prediction

## Tech Stack

- React
- TypeScript
- TailwindCSS
- Express.js
- MongoDB
- Socket.io
- JWT
